# GryEngine

A C++ game engine developed as a hobby :)
This project is ongoing, and it's primary purpose is C++ / Game Engine Design practice.

## Created in:
- Visual Studio 2017

## Utilizing:
- C++
- GLFW
- GLEW
- OpenGL
